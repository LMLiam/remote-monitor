# Support

Use the channel that matches your request.

| Request | Channel |
| --- | --- |
| Usage question or setup help | Start a [GitHub Discussion](https://github.com/LMLiam/remote-monitor/discussions). |
| Reproducible bug | Open a [bug report](https://github.com/LMLiam/remote-monitor/issues/new?template=bug_report.yml). Include commands, flags, local terminal details, and remote host context. |
| Feature request | Open a [feature request](https://github.com/LMLiam/remote-monitor/issues/new?template=feature_request.yml). Describe the user need and any Linux, SSH, terminal, or GPU assumptions. |
| Sensitive security report | Use [private vulnerability reporting](https://github.com/LMLiam/remote-monitor/security/advisories/new) and read [SECURITY.md](SECURITY.md). Do not put secrets, hostnames, credentials, exploit details, or private logs in a public issue. |

There is no paid support channel and no guaranteed response time.
